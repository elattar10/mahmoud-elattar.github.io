About Me

Turning raw data
into real-world value

I'm a Data Engineering student at Zagazig University, passionate about building the infrastructure that makes data useful. I specialize in designing and implementing robust pipelines that handle data at scale.

I completed the DEPI Microsoft Data Engineer program — a competitive government-backed initiative — where I built a real-world Smart City IoT pipeline as my capstone project.

My stack spans from Python and SQL for scripting and data manipulation, to Apache Kafka, Spark, and Hadoop for big data processing, all containerized with Docker and orchestrated via Airflow.

When I'm not building pipelines, I'm expanding my German vocabulary or exploring new areas in cloud data engineering.
